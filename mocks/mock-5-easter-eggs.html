How these compose

Recommended set if you want a unified visual language

Picking ones that don't overlap and read as a single designed experience across the session:

  1. Brewing coffee loader — replaces both Phase B and Phase D research-wait copy.
  2. Departures board — runs continuously through Phase E so the user can see drafting progress.
  3. Concierge wax seal — stamps onto the Decisions Made card and survives in the saved markdown.
  4. Vintage postcard — alternate skin for Decisions Made (pick one of seal-or-postcard, not both).
  5. Star burst — final shimmer after Decisions render. Cheap, satisfying.

If you want one more, the Welcome cocktail after Phase A is the most distinctive standalone moment.
